# Optional graphics

Drop your own PNG/JPG/WebP files in this folder using these names:

- `bcbc-logo.png` — optional logo shown at the top-left of the hero area
- `bcbc-header.png` — optional wide header/banner shown in the hero area
- `bcbc-footer.png` — optional footer/banner shown above the credits

Recommended sizes:

- `bcbc-logo.png`: square or transparent PNG, around 512x512
- `bcbc-header.png`: wide banner, around 1600x400
- `bcbc-footer.png`: wide banner, around 1600x250 to 1600x400

If a file is missing, the page automatically hides that image slot.
